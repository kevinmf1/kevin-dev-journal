# 📓 Kevin Dev Journal

My personal dev journal — documenting development notes, learnings, and progress.

🔗 **Live site**: [kevinmf1.github.io/kevin-dev-journal](https://kevinmf1.github.io/kevin-dev-journal/)

## 🛠️ Tech Stack

- [Docusaurus 3](https://docusaurus.io/) — Static site generator
- [React](https://react.dev/) — UI framework
- [GitHub Pages](https://pages.github.com/) — Hosting

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v20.0 or above

### Installation

```bash
npm install
```

### Local Development

```bash
npm run start
```

Starts a local development server at `http://localhost:3000/kevin-dev-journal/`. Changes are reflected live without restarting the server.

### Build

```bash
npm run build
```

Generates static files into the `build` directory.

### Serve Production Build

```bash
npm run serve
```

Serves the production build locally for testing.
